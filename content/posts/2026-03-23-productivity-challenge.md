This challenge came about from having my ISP do a sneaky shutdown of my internet
due to technical issues (they claim, but I suspect it was malice).

I found my self being rather productive during this time and so I told the wife
about this.

We then decided to try this out the coming Saturday and turned the modem off
between 06:00 and 13:00.

As a result we got so much done we decided to do it again the next day, but for
a shorter period, so we turned it off between 06:00 and 11:00.

The TL;DR is this: 
Turn off your modem / router / fibre-magic-box / FrizBox for a few hours and log your
productivity.

The goal of this is just a bit of fun but it can also help with getting things done.

## The rules

### Optional rules

* Allow a singular device for information lookup (e.g a phone with mobile data)
* Don't allow any screen time
 
## Run time

This challenge runs for two days (feel free to repeat it).

- Day 1: seven hours 
- Day 2: five hours

Write down the tasks you have worked on or even finished during this
time window and post your summary each day on your Discord

The format of the post:

```
Day #
Tasks: 
    * list
    * of
    * tasks
Did you manage: yes / no (only answer "no" if you gave up)
```

E.g
```
Day 1
Tasks:
    * Wrote some Rust
    * Did some CAD
    * Invented a card game
Did you manage the entire time: yes
```

Good luck!
